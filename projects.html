01
basil
Smart Pantry Web App · Hackathon
September 2025

Built at Georgia Tech's hackathon, basil solves a real problem: keeping track of what's in your fridge before it expires. I architected the backend — a Flask/SQLite/SQLAlchemy stack serving a React frontend — and tackled the two hardest parts: wrangling the USDA food database and building an AI recipe generator that produces consistent, accurate nutritional data.

The AI integration required real prompt engineering: I built a custom Mosaic/OpenAI client that forces structured JSON output (including macros), which the backend parses and validates. Authentication uses Amazon Cognito with proper session management — something we learned at an AWS workshop the same weekend and implemented from scratch in under 24 hours.

  • Parsed 8,000+ USDA food database entries from raw CSV into a queryable SQLite/SQLAlchemy schema
  • Engineered structured JSON prompts for consistent, macro-accurate AI recipe output
  • Implemented Amazon Cognito auth with refresh token handling and session expiry edge cases
  • Collaborated on React frontend for inventory management and recipe browsing

02
Malware Reverse Engineering
Binary Analysis · Ongoing
Ongoing

Using IDA Pro to perform deep static analysis on real-world malware samples, annotating x86 assembly line-by-line to reconstruct program logic, document exploit mechanisms, and understand how malicious software actually operates at the binary level.

Each sample teaches something different. SQLSlammer showed me how a 376-byte worm can spread globally using a single UDP packet and a stack-based buffer overflow. Michelangelo revealed classic boot sector infection. DOS7 showed stealth via interrupt hooking and memory-resident code.

  • Analyzed W32.SQLSlammer: buffer overflow exploit chain, null-free shellcode, PEB traversal for runtime API resolution, and jmp esp stack exploitation
  • Analyzed Michelangelo: boot sector infection, MBR/partition table overwrites, and payload trigger logic
  • Analyzed Virus.DOS.Dos7: interrupt hooking, memory-resident behavior, and stealth techniques
  • Ongoing — adding Lucius and additional samples
IDA Prox86 AssemblyStatic AnalysisExploit AnalysisShellcode
